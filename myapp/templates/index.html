Acknowledgements

To access the TMDB API I used the TMDBsimple wrapper, a very helpful tool. Furthermore it is simply great that TMDB provides this API to share their data with people like me. A big help for adding tables to the dashboard was Derek Eders tutorial about converting a CSV to HTML Table.

Data

Since 2016, I have my own diary with information about movies, operas, books and concerts, that I watch, read and attend. Each item is also evaluated (0-100) and I write down who accompanied me on the event. I also add data from The Movie Database to compare my own evaluation with theirs. Furthermore, it gives me the possibility to learn more about the movies I like and eventually prepare a movie recommender. For operas, I have to check if there will be a recommender - or better if there is anybody in need of an opera recommender ;)
